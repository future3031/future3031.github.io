指導教授：王珮瑜教授

組員：家瑩、思吟、健翔、冠廷

我想學習數位內容相關領域的系所，在大四時都會需要完成一個四年的成果報告－畢業專題。

然而嘉大數位系也不意外地希望學生可以透過四年所學的知識和能力來完成一項專案。

Overview

然而系上的時間安排是三下前的寒假就是各組找指導教授的戰爭惹，需要先與組員達到一個主題共識，再一同找相關領域的指導教授。

製作將近一年的時間，在四上的期末進行成果發表，四下在辦理校內與校外展覽。

由於我們組希望可以製作故事性和闖關類型的手遊，但一開始其實也沒有很確定要做什麼主題，內容是甚麼。

不管是內部討論，還是與指導教授討論了多次，我們仍想不到甚麼樣的主題有意義性、有趣、且是我們自己蒐集資料也不排斥的呢?

突然，組員一個蹦出”那我們做跟民主相關的如何?”

接者，大家開始天馬行空的將畢專藍圖畫出，從民主到歷史中的重大事件，並從中找出美國南北戰爭、英國工業革命、德國納粹、蘇聯共產體制等等一些其實很重要，但卻從未在高中歷史課本中出現的內容。

Project Background

我們覺得歷史是文化中很重要的一部分，若沒有歷史，我們無法知道在幾千年前人民是有分階級的的，無法知道前人是怎麼推翻獨裁政治的。但深入挖掘後，我們發現很多事件中的小故事，我們竟然不知道，高中的歷史從未講過，所以希望能透過此專題讓學習者(玩家)瞭解不同面向的歷史故事。

主要遊戲的內容是以高中歷史課本中的西洋歷史，希望能透過這些重要事件啟發學習者(玩家)對歷史的好奇心與興趣。

  • 交付項目：Unity製作手遊、網頁、小說電子書
  • 工作內容：故事腳本(四個遊戲故事)、美術設計(人物及場景)、程式設計(遊戲及網頁)

我主要負責是故事腳本與美術設計(人物)。

Game Story

我們設計故事裡有一位名叫"巴倫"的男主角，有一天放學回家，在家附近發現一間從來沒看過的房子，充滿好奇心的他決定走進屋裡一探究竟，結果在屋內看到一位遍體麟傷的老人倒在地上，老人希望巴倫能夠到英國、美國、德國、匈牙利這四個國家尋找神秘寶石，來幫助他回到原本的世界。

因此，巴倫透過老人的神秘力量，來到了不同年代的各地區，經歷了四個歷史事件尋找寶石。

Introduction

首先，我們先把每個任務細分個子任務，並安排每位組員負責的工作，以及時程，下圖是我們當初畫的工作結構圖。

由於，從三下開始，固定會有期中和期末的present，這也是促進組別的專案進度，總共會有四次的進度報告。

因此，我們以過去課堂的製作經驗，先粗略的分配每次報告的時程規劃，加上我們四人在三下後的暑假皆有實習，因此，在暑假之前，我們希望在美術上有所進度，讓程式可以順利在暑假時有進度。

但我們都把自己想的太美好，在暑假時進度嚴重的落後，導致我們後期程式的同學極度痛苦。

然而四上算是衝刺期，開始去兌現當初第一次的妄言(哈)，加上我們也希望製作出來的手遊是可以讓高中生實際遊玩的，因此找了7位的高一生進行測試，並透過問卷調查的方式瞭解使用者體驗。

Reflection

想要先感謝當初強大的組員們，從一開始的專案構思，到最後的展覽呈現，雖然很可惜的沒有一起到放視大賞被別人電，但這一路上的相扶相持都讓我永生難忘。

尤其想感謝家瑩組長，在美術上給了我很多的建議與指導，也發現自己美術上的不足，能力上仍不夠。

如今已過了兩年多，回想起當初各自為了自己工作進度所熬的夜，每次開會時的認真態度，以及與老師的幹話討論，都是大學時難忘的經驗。

雖然現在回想起當初在成果報告時自己的態度很衝動，以及當初的不理解委員們的建議到現在逐漸瞭解專案的嚴謹性，都是當初自己沒有仔細思考的地方。

但，我想這就是成長吧！

沒有畢專的經驗、沒有被砲轟的經驗、沒有與一群人為了一個目標努力完成的經驗，或許就沒有現在的我。

雖然難免有些後悔的地方，但我想FINCH不僅僅是歷史的傳承，更是我們成長的基底。

記得！在人生的道路上，我們都是所向無敵的！
